#Zombei Shooting Range Game

The script.js file is the main JavaScript file responsible for running a Scratch/TurboWarp project in a web browser. It contains the code required to load, execute, and manage the project, including graphics rendering, user interactions, and program logic.
Purpose
The primary purpose of this file is to:
Run Scratch/TurboWarp projects.
Manage sprites, variables, and events.
Handle user input such as keyboard and mouse actions.
Render graphics using WebGL.
Load additional modules and extensions when required.
Main Features
1. Module Loading
The file uses Webpack technology to organize and load different JavaScript modules efficiently.
2. Scratch Virtual Machine
It includes the Scratch VM, which executes Scratch blocks and controls project behavior.
3. Graphics Rendering
The script uses WebGL and TWGL libraries to display animations, sprites, and visual effects.
4. Variable Management
It manages:
Variables
Lists
Broadcast messages
User-defined data
5. Event Handling
The script responds to:
Keyboard input
Mouse clicks
Broadcast events
Timer events
6. Extension Support
It allows external extensions to add new functionality and custom blocks.
Technologies Used
JavaScript
Webpack
Scratch VM
TurboWarp
WebGL
TWGL Graphics Library
Advantages
Fast execution of Scratch projects.
Supports advanced graphics.
Modular and scalable architecture.
Compatible with modern web browsers.
